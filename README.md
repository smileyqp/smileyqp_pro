#### 项目描述:

- 此项目为前后台分离的招聘的SPA(单页应用),保罗前段应用和后端应用
- 包括用户注册/登录,实时聊天等模块
- 前段使用:React全家桶+ES6+webpack(打包工具)等技术
- 后端:使用Node+express+mongodb+socketIO等技术
- 采用模块化,组件化,工程化的模式开发
